# Open Measurement Interface Definition (OMID) API

[![IAB Tech Lab](https://www.iabtechlab.com/wp-content/uploads/2019/04/IABTechLab_Logo_2xWidth-1.png)](https://iabtechlab.com)

## About

The OMID API facilitates third-party viewability and verification measurement for ads served to mobile app, web video, and CTV environments. It enables standard communication from any supporting Ad SDK to any supporting measurement tag, removing the need for multiple integrations specific to each Measurement Provider.

**Current Version:** 1.5 (June 2024)
